# 📊 Python Data

Este repositório foi criado com o objetivo de reunir meus estudos e práticas utilizando Python para análise de dados e estatística.

Aqui irei adicionar notebooks, exercícios e exemplos envolvendo bibliotecas muito utilizadas na área de dados, principalmente:

- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- TensorFlow

O foco principal deste repositório é servir como um ambiente de aprendizado e prática, onde irei estudar conceitos importantes de manipulação de dados, estatística e machine learning.
